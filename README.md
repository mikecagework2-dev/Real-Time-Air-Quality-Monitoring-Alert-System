# Real-Time Air Quality Monitoring & Alert System

## Project Overview
A web-based application for monitoring real-time air quality data with alerts and health recommendations.

## Features
- Real-time air quality monitoring
- Location-based tracking
- Alert notifications
- Health recommendations
- Interactive data visualizations
- Location comparison

## Tech Stack
- **Frontend**: React.js, Chart.js
- **Backend**: Flask (Python)
- **Database**: SQLite
- **API**: OpenWeatherMap Air Pollution API
